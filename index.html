Mes Compétences

Python

Python

Python est un langage de programmation interprété, multi-paradigmes et multiplateformes. Il favorise la programmation impérative structurée, fonctionnelle et orientée objet.

MySql

MySql

MySQL est un système de gestion de bases de données relationnelles (SGBDR) libre, multi-utilisateurs et multi-tâches. Il est développé, distribué et supporté par Oracle Corporation.

Figma

Figma

Figma est un logiciel de conception graphique et de prototypage interactif en ligne. Il permet de créer des maquettes de sites web et d'applications mobiles.

Html 5

Html 5

HTML5 est la cinquième et dernière révision majeure du HTML standard, définie par le W3C. Il s'agit d'une norme du W3C pour le langage de balisage HTML.

Css 3

Css 3

CSS3 est la troisième révision majeure du langage informatique de feuilles de style en cascade. Il a été publié en décembre 2014 par le World Wide Web Consortium (W3C) comme une recommandation de standard.

JavaScript

JavaScript

JavaScript est un langage de programmation de scripts principalement employé dans les pages web interactives mais aussi pour les serveurs. Il est principalement connu sous le nom de JavaScript, mais il existe une variété de noms pour désigner ce langage.

Git Github

Git Github

Git est un logiciel de gestion de versions décentralisé. Il permet de gérer les versions d'un projet informatique, en particulier lors du développement d'un logiciel.

Vs Code

Vs Code

Visual Studio Code est un éditeur de code source développé par Microsoft pour Windows, Linux et macOS.
